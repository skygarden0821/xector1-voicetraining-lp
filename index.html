LESSON'S FEATURE
XECTOR1 の特徴
FEATURE 01

担当制のマンツーマンレッスン

あなた専属の講師が、レベルや目標に合わせたカリキュラムを設計します。基礎からボイトレしたい方も、作品作りから始めたい方も安心。担当制だからアドバイスが一貫して、成長スピードが違います。あなただけのカリキュラムなので質問も気軽にでき、レッスンに集中できます。

FEATURE 02

ボカロ曲をかっこよく歌うコツをお伝えします

ボカロ曲は音域も広く、テクニックを使った表現力も多彩です。まずは好きな歌い手がどのように歌っているのかを一緒に聞きながら、ウィスパーボイス、がなり、エッジ、ビブラートなどテクニックがどのように使われているかを確認。その上でゆっくりとそのテクニックを入れて歌えるようにしていきます。

FEATURE 03

ボイトレもしっかりレッスンします

表現力をつけるためには基礎も大切です。腹式呼吸や声を響かせる身体の使い方を1から学んでいきましょう。レコーディングをすると自分の苦手なところが気になり、再度ボイストレーニングをしたいという気持ちが出てきます。それは前に比べて下手になったのではなく上達した証です。

FEATURE 04

歌ってみた動画を投稿してみたい

歌ってみた動画を作る工程は、①レコーディング → ②ミックス・マスタリング → ③動画撮影 → ④動画編集。まずは講師と一緒に歌ってみた動画を作ってみましょう。2本、3本と作っていくと作り方に慣れ、自分でもチャレンジできるようになります。

FEATURE 05

ミックス・マスタリングを学びたい

ピッチ修正、イコライザー、コンプレッサーなど複雑に思えるかもしれませんが、講師と一緒に曲を仕上げながら行うので理解がしやすく、すぐに使える技術になります。ミックスマスタリングを覚えると歌い手としてレベルアップできます。

FEATURE 06

SNS発信・コーチングまでカバー

歌がうまくなるだけでは終わりません。YouTube・TikTok・InstagramなどSNSでの発信方法、フォロワーの増やし方までプロがアドバイス。アウトプットすることで自分が目指す地点との差がわかり、何をすればいいかが明確になります。

FAQ

Q全くの初心者ですが大丈夫ですか？+
はい、大丈夫です！生徒さんの多くが未経験からスタートしています。あなたのレベルに合わせたカリキュラムを組みますので、基礎の基礎から安心して学べます。
Q必要な機材はありますか？+
まずはスマートフォンやPC、イヤホンがあればOKです。レコーディングに進む際は、講師からおすすめの機材をご紹介します。無理に高い機材を揃える必要はありません。
Qレッスンのスケジュールは柔軟に変更できますか？+
はい。講師と直接スケジュールを調整できるので、シフト制のお仕事や学校の予定に合わせて柔軟にレッスン日時を決められます。
Q途中で退会はできますか？+
はい、いつでも退会可能です。長期契約の縛りはありません。月単位でのご契約となりますので、お気軽にお試しください。

60分間のオンラインレッスンです。前半にカウンセリングで目標やお悩みをヒアリングし、後半に実際のボイストレーニングを体験していただけます。
